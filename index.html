Make It Stick

MakeItStick is an organizer, its main objective is to categorize tasks/todos in logical way. The general structure of a project is a tree design where every element (todos or categories) will have a unique father, but each father could have endless children. The project will count on two basic elements “todos and categories”. The categories are the container that will bear the todos or other categories called “subcategories”. On the other, the todos will have a particular father category and also features like levels of importance and tags, that will help to share concepts with other todos and help the search of the todos. This project was built with the well-known TodoList Project in mind. The wireframe and general design were built using tools like Adobe Photoshop, Adobe Illustrator and Balsamiq.

At the last stage, was added all the different user interactions via JavaScript and jQuery and the element menu, one of the most interesting features of the project, which permit the interaction with a particular element (todo or category) to be deleted, created, moved or edited [CRUD]. Almost all of the interactions were powered by AJAX requests and PHP calls. The project also contains a search form, that could filter results based on category name, todo name or tag name; all accomplished with PHP and SQL. And the last feature that was added was the registration form which permit to create and control new users for the application.

Dorotea

The main purpose of Dorotea Collection was to bring to the web the beauty of magazine design. In this website, fonts and photos play a crucial role giving a sense of elegance and serenity to the entire product. As well as the colors used to match perfectly with each piece of the collection.

The wireframe and basic design were created with Adobe Photoshop and Balsamiq. The styles were added using Sass preprocessor and Bootstrap library. CSS-Grid also played a huge role on the main design making possible the principal collage of the website; this was also accomplished with JavaScript and jQuery.

Bootstrap Modals and Carousels were used to display the gallery products. One of the most interesting features I learnt thanks to this project was CSS-grid layout.

Blue Jaxon

This project found inspiration on the Amazon Marketplace website, having as a main objective the distribution of products by giving descriptions, images and sellers information.

The website follows the e-commerce design and was created on the platform WordPress.

The first stage of this project was the design of the website, a three columns layout with a flexbox display of the products created with Saas preprocessor and Bootstrap. The design of the template was created on a child-theme along with all the hierarchy of the website (front page, singular page, archive page, etc.).

Blue Jaxon also uses an AJAX plugin to help calling new products while scrolling on the page. Each product gets redirected to its own page on Amazon.com, ready to be bought.
